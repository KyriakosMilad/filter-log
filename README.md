# filter-log
simple go cli tool to filter log file.

## Install
1. ```  git clone https://github.com/KyriakosMilad/filter-log ```
2. ```  cd filter-log ```
3. ```  go install . ```
4. Now you can use filter-log command from anywhere.

## Usage
    filter-log -path <path_to_the_log_file> -level <level_to_search>

## Help
    filter-log -help
